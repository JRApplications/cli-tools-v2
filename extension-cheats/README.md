# Extension Cheats

> **Note:** "Cheats" is a loose term — these are not actual cheats. They are undocumented extension configurations that can be added to your project, similar to what you would normally set up through the app dashboard.

## What Are Extension Cheats?

Extension cheats are unofficial, undocumented properties and configurations that can be added directly as an extension rather than through the app dashboard. They achieve the same result but bypass the usual UI workflow.

## Important

- These are **not officially documented** and may change or break without notice
- Use at your own risk in production environments
- When possible, prefer the official app dashboard for stability
